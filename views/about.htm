How it works

This website is powered by Angular.JS and Google Sheets. We query a Google Sheet with lots of live data in it every time a page is loaded, using an XMLHttpRequest();. We then use algorithms to compile the data into neatly laid-out tables, cross-referencing each other for easy navigation. The website is styled by Bootstrap. It's made to be as compact as possible so that it loads quickly on phones and doesn't use up 'too much data'. It's entirely responsive, and was tested mobile-first.
