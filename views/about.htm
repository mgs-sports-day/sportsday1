About

This website was developed to allow spectators of the Manchester Grammar School's annual Senior Sports Day to view results, events, and competitors in these events. It is a collaborative effort between the Computer Science and PE departments of the school, the staff responsible in each department being Mr D Millington and Mr T Grainger respectively. It is hoped that students, parents and staff will find this website beneficial. It works by fetching score and competitor data from a Google Drive spreadsheet, and displaying it in an easy-to-use and clear fashion.

How it works

This website is powered by AngularJS and Google Sheets. We query a Google Sheet with lots of live data in it every time a page is loaded, using an XMLHttpRequest();, using the Google Sheets API v4. All calculation of scores and results occurs in Google Sheets: not on the website. We then use various algorithms to compile the data into neatly laid-out tables, cross-referencing each other for easy navigation. The website is styled by Bootstrap. It's made to be as compact as possible so that it loads quickly on phones and doesn't use up 'too much data'. It's entirely responsive, and was tested mobile-first.

Name of cookie Purpose Time stored for
stopSurveyAlert Determines if we should show you a message asking you to fill out a user experience survey. When you first visit the site, this cookie is set to false. cookieconsent_status Determines if you have agreed to the cookies and privacy statment, so we do not show it to you again. This cookie is set to dismiss when you click "I agree" on the consent banner. 24 hours

Privacy

This site collects no personal data about its users.

This site uses the service Google Analytics to collect non-personal data about those who visit it, such as (but not limited to) their web browser, mobile network, and country. Google Analytics works by placing cookies (see below) on your device to identify each user to them. Google stores data about your visit for 26 months, after your most recent visit to the site. Data collected by Google Analytics is visible only to the site's developers. By using this site, it is assumed that you agree to this. We advise that all users read the Google Analytics Privacy and Cookie policy here: http://www.google.com/policies/privacy/partners/. If you do not want this information to be collected by us, please install the official Google Analytics opt-out extension here: https://tools.google.com/dlpage/gaoptout.
